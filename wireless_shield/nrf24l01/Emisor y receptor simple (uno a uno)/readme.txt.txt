El código para hacer funcionar el sistema fue obtenido de la siguiente url: http://www.prometec.net/nrf2401

En este sktech se establecio un nodo emisor y otro nodo receptor. El emisor manda datos númericos de 0-255 y el receptor recibe este
dato y lo muestra en el puerto serial. Para lograr esto se establece un pipe o canal de comunicación, donde el emisor envia los datos
y el receptor escucha.
